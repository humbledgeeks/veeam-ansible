# veeam-ansible

Ansible playbooks for Veeam Backup & Replication automation.

## Contents

| Playbook | Purpose |
|----------|---------|
| `veeam-job-status.yml` | Gather and report Veeam backup job status |

## Prerequisites

- Ansible 2.14+
- Access to Veeam Backup Server (REST API or WinRM)

## CI/CD

All PRs are validated by ansible-lint, secret scan, and header compliance checks.
